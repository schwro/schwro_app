Osadzaj formularze z Church Manager na swojej stronie WordPress za pomocą prostego shortcode.

== Description ==

Church Manager Forms pozwala łatwo osadzać formularze stworzone w aplikacji Church Manager na Twojej stronie WordPress.

**Funkcje:**

* Prosty shortcode do osadzania formularzy
* Blok Gutenberg dla łatwej edycji wizualnej
* Widget Elementor
* Responsywne osadzanie
* Automatyczne dostosowanie wysokości (opcjonalnie)
* Wsparcie dla wielu formularzy na jednej stronie

**Użycie:**

1. Zainstaluj i aktywuj plugin
2. Przejdź do Ustawienia → Church Manager Forms
3. Wprowadź URL swojej aplikacji Church Manager
4. Użyj shortcode `[church_form id="TWOJE_ID"]` lub bloku Gutenberg

== Installation ==

1. Pobierz plugin i rozpakuj go
2. Skopiuj folder `church-manager-forms` do `/wp-content/plugins/`
3. Aktywuj plugin w panelu WordPress
4. Przejdź do Ustawienia → Church Manager Forms i skonfiguruj URL aplikacji

== Frequently Asked Questions ==

= Gdzie znajdę ID formularza? =

ID formularza znajdziesz w aplikacji Church Manager. Przy każdym formularzu jest opcja "Osadź na stronie", która pokazuje ID oraz gotowe kody do osadzenia.

= Czy mogę osadzić wiele formularzy na jednej stronie? =

Tak, możesz używać wielu shortcode'ów z różnymi ID formularzy na jednej stronie.

= Czy formularz działa na urządzeniach mobilnych? =

Tak, formularze są w pełni responsywne i dostosowują się do rozmiaru ekranu.
